CSS Float Property

This page demonstrates how floating elements affect layout, text wrapping, clearing behavior, and common float-based layouts.

float: right

This image is floated to the right. The surrounding text wraps naturally around it, adjusting as the screen size changes.

float: left

Here the image floats to the left, allowing text to flow on the right side. This technique is commonly used for articles and blog layouts.

float: none (default)

With no floating applied, the image stays in the normal document flow.
