CSS display: inline-block

The inline-block value combines inline flow with block-level sizing and spacing capabilities.

Why inline-block is useful

Inline elements ignore width and height, while block elements break onto a new line. Inline-block allows elements to stay on the same line and respect width, height, and vertical margins.
